## 📖 Examples

### Basic Usage
```bash
# Interactive mode / 交互模式
anyig

# Generate specific template / 生成特定模板
anyig -t gitignore

# Multiple selection / 多选模式
anyig -m
```

### Configuration Example
```json
{
  "defaultTemplates": ["gitignore", "npmignore"],
  "outputDir": "./generated",
  "autoBackup": true,
  "confirmOverwrite": false
}
```
